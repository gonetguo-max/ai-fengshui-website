## 🚀 **部署指南**

### 📋 **生产环境准备**

#### 1. **环境配置**
```bash
# 1. 设置环境变量
cp example.env .env
# 编辑 .env 文件，配置生产API密钥

# 2. 安装依赖
npm install --production

# 3. 启动服务
npm start
```

#### 2. **服务器要求**
- **Node.js**: 16.0.0+
- **内存**: 最低512MB，推荐1GB+
- **存储**: 最低1GB（用于图片上传）
- **网络**: 稳定的DeepSeek API连接
